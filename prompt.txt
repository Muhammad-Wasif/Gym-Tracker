CRITICAL DESIGN MANDATE — READ THIS BEFORE WRITING A SINGLE LINE:

This UI must look like it was designed by a senior product designer at a premium
fitness company — not generated by an AI. That means:

  - NO generic purple-on-white gradients
  - NO Inter or Roboto fonts
  - NO flat cards with grey borders that look like a Bootstrap tutorial
  - NO uniform padding everywhere that looks like a template
  - NO icons that are just emoji or basic shapes

The aesthetic is: DARK ATHLETIC LUXURY
  Think: Nike Training Club meets Bloomberg Terminal meets Notion dark mode
  Deep navy/slate backgrounds, electric blue accents, sharp typography
  Asymmetric layouts, oversized numbers, data-forward design
  Animations that feel physical — spring-like, weighted, intentional

Rules you must follow without exception:
  - Zero comments in any code file
  - Zero placeholder pages — every page is fully functional
  - Zero hardcoded data — every screen reads from Service classes
  - MVVM pattern strict — zero business logic in code-behind
  - Every page has a ViewModel
  - All services instantiated with new (no DI framework)
  - All Service methods are synchronous — no async/await
  - No EF Core, no repositories — UI only talks to Service classes
  - Animations are coded in C# using Storyboard, not XAML only
  - Every interactive element has a hover state and a click state

public partial class App : Application
{
    public static Person? CurrentUser { get; set; }

    protected override void OnStartup(StartupEventArgs e)
    {
        base.OnStartup(e);
        var login = new LoginWindow();
        login.Show();
    }

    public static void NavigateToMain()
    {
        var main = new MainWindow();
        main.Show();
        foreach (Window w in Current.Windows)
            if (w is LoginWindow) { w.Close(); break; }
    }

    public static void Logout()
    {
        CurrentUser = null;
        var login = new LoginWindow();
        login.Show();
        foreach (Window w in Current.Windows)
            if (w is MainWindow) { w.Close(); break; }
    }
}

GoalToColorConverter         : GoalName string → SolidColorBrush (goal color map)

PercentageConverter          : (double actual, double target) → double (0.0 to 1.0)
InitialsConverter             : "Muhammad Wasif" → "MW"
DateToGreetingConverter      : DateTime → "morning" / "afternoon" / "evening"
MacroToColorConverter        : "Protein" → ProteinBrush, "Carbs" → CarbsBrush, "Fat" → FatBrush

All major animations must be implemented this way — not with generic WPF transitions.

─── Page entry animation (call in Page.Loaded) ──────────────────────────────────

private void AnimateIn(UIElement element, double fromY = 20, double delay = 0)
{
    element.Opacity = 0;
    var tt = new TranslateTransform(0, fromY);
    element.RenderTransform = tt;

    var sb = new Storyboard();

    var fadeIn = new DoubleAnimation(0, 1, TimeSpan.FromMilliseconds(350));
    fadeIn.EasingFunction = new CubicEase { EasingMode = EasingMode.EaseOut };
    fadeIn.BeginTime = TimeSpan.FromMilliseconds(delay);
    Storyboard.SetTarget(fadeIn, element);
    Storyboard.SetTargetProperty(fadeIn, new PropertyPath(UIElement.OpacityProperty));

    var slideIn = new DoubleAnimation(fromY, 0, TimeSpan.FromMilliseconds(350));
    slideIn.EasingFunction = new CubicEase { EasingMode = EasingMode.EaseOut };
    slideIn.BeginTime = TimeSpan.FromMilliseconds(delay);
    Storyboard.SetTarget(slideIn, tt);
    Storyboard.SetTargetProperty(slideIn, new PropertyPath(TranslateTransform.YProperty));

    sb.Children.Add(fadeIn);
    sb.Children.Add(slideIn);
    sb.Begin();
}

─── Button hover / click animation ─────────────────────────────────────────────

In button style trigger (ControlTemplate):
  MouseEnter: ScaleTransform 1.0 → 0.98, 80ms
  MouseLeave: ScaleTransform 0.98 → 1.0, 120ms
  MouseDown:  ScaleTransform → 0.96, 50ms
  MouseUp:    ScaleTransform → 1.0 with BackEase (spring), 200ms

─── CountUp animation (StatCard, AnimatedCounter) ──────────────────────────────

private void AnimateCountUp(TextBlock tb, double target, string format = "0.0")
{
    var anim = new DoubleAnimation(0, target, TimeSpan.FromMilliseconds(800));
    anim.EasingFunction = new CubicEase { EasingMode = EasingMode.EaseOut };
    anim.FillBehavior = FillBehavior.Stop;

    double current = 0;
    anim.CurrentTimeInvalidated += (s, e) =>
    {
        if (s is AnimationClock clock && clock.CurrentProgress.HasValue)
        {
            current = target * clock.CurrentProgress.Value;
            tb.Text = current.ToString(format);
        }
    };
    anim.Completed += (s, e) => tb.Text = target.ToString(format);

    tb.BeginAnimation(null, null);
    var clock2 = anim.CreateClock();
    tb.ApplyAnimationClock(OpacityProperty, null);
    clock2.Controller?.Begin();
}

─── List stagger animation ──────────────────────────────────────────────────────

private void AnimateListItems(Panel container, double delayMs = 60)
{
    int i = 0;
    foreach (UIElement child in container.Children)
    {
        AnimateIn(child, fromY: 16, delay: i * delayMs);
        i++;
    }
}

─── Line chart draw animation ──────────────────────────────────────────────────

After drawing the Polyline on Canvas:
  Set StrokeDashArray = new DoubleCollection { totalLength, totalLength }
  Set StrokeDashOffset = totalLength
  Animate StrokeDashOffset from totalLength to 0 over 1200ms, CubicEaseOut
